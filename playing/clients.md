## **TuffClient**

**Main Developers**
- shaun
- justatypicalpotato

**Main Features**
```
- Below y0 - loads blocks below y = 0 (requires server plugin)
- ViaVersion Items - loads item textures for 1.13+ items
- ViaVersion Entities - loads models and textures for 1.13+ mobs
- ViaBlocks (COMING SOON) - loads 1.13+ blocks in the world
- Select positions of all GUI modules

```

## PixelClient

**Main Developers:**
[devjakob](https://github.com/ProJakob)

**Features:**
```plaintext
Features:
  - Support for community translations
  - Highly customizable
  - Full ARGB and Chroma color support
  - Regular updates
  
Versions:
  - 1.8.8 (PixelClient 2.1.4)
  - 1.12.2 (PixelClient 2.1.4)

Client Settings:
  - Uncap HUD (Renders the HUD at higher FPS to make animations smoother)
  - Minimal Bobbing (Disables Chrosshair shake when View Bobbing is enabled)
  - Left Hand
  - WSNoDelay (Attempts to lower ping in some situations)
      EXPERIMENTAL: Edits packet handling

Mod List:
  - Armor Status
      Shows stats about your armor
  - Block Overlay
      Edit the block overlay
      Settings:
        - Color
        - Width
  - Brightness (aka. Fullbright)
      Light up the world
      Settings:
        - Boost
  - ClientSniffer (aka. CheaterSniffer)
      Detect players clients
      Settings:
        - Clients: Comma seperated list
  - CPS
      Show your clicks per second
      Settings:
        - Color
        - Background Color
  - FPS
      Show your FPS in game
      Settings:
        - Color
        - Background Color
        - Animated
  - Free Look
      Look around your player freely
      Settings:
        - First Person
        - Return on release
  - Hitboxes
      Customize entity hitboxes
      Settings:
        - Outline
        - Headbox
        - Aim
  - Hurt Cam
      Modify the camera shake
      Settings:
        - Multiplier
  - Hurt Color
      Edit the damage tint
      Settings:
        - Color
  - Keystrokes
      Shows your inputs
      Settings:
        - Mode (WASD, WASD_MOUSE, WASD_SPRINT, WASD_SPRINT_MOUSE, WASD_JUMP, WASD_JUMP_MOUSE, WASD_JUMP_MOUSE_CPS)
        - Animated
        - Aim
  - No Dynamic FOV
      Prevents FOV changes
  - Ping
      Show your Ping in game
      Settings:
        - Color
        - Background Color
  - Potion (aka. Potion Timer)
      View your active potions
  - Rendering (aka. FPS Boost)
      Control parts of the games rendering
      Settings:
        - Rain: Should rain be rendered?
        - Chat background: Should the chat background be rendered?
        - Own nametag: Should your nametag be rendered?
  - Speed
      Display your speed
      Settings:
        - Color
        - Background Color
  - Time Changer
      Change the time
      Settings:
        - Time value
  - TNT Timer
      Shows a countdown above ignited TNT
      Settings:
        - Fuse modifier
  - Toggle Sprint
      Toggle sprint?!
```
